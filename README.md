## Description
This is a project using Astro as a static single generator.
Inside of this project, you'll see the following folders and files:

```
/
├── public/
│   └── assets/
│   └── styles/
├── src/
│   └── components/
│   ├── layouts/
│   │   └── Layout.astro
│   └── pages/
│       └── index.astro
└── package.json
```
